FullStack: Angular + Java + MySQL

Neste curso, você vai aprender a desenvolver aplicações web completas, desde o front-end até o back-end, usando as tecnologias Angular, Java e MySQL. Você vai criar componentes reutilizáveis, rotas dinâmicas, formulários reativos, autenticação e autorização com tokens JWT, tabelas de listagem de dados com paginação e ordenação, entre outros recursos avançados.

O curso é composto por 5 seções, 61 aulas e duração total de 7h 37m. As aulas são 100% práticas e você vai acompanhar o passo a passo da construção de uma aplicação web completa. Além disso, você vai ter acesso aos códigos-fonte dos projetos e aos exercícios propostos para fixar o conteúdo.
